The "current-bit" property

  "current-bit"              gboolean              : Read

The "first-index" property

  "first-index"              guint                 : Read / Write

The first index to select pattern.

Allowed values: <= 256

Default value: 0

The "offset" property

  "offset"                   guint                 : Read / Write

The offset within the pattern.

Allowed values: <= 65535

Default value: 0

The "port" property

  "port"                     AgsPort*              : Read / Write
